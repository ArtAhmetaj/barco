# Network formats

All network encoding is big-endian.

## Consumer poll response

```
+------------------------------------------------------------------------------------------+
| items length (uint16)                                                                    |
+------------------------------------------------------------------------------------------+
| response items                                                                           |
| +--------------------------------------------------------------------------------------+ |
| | item                                                                                 | |
| | +---------------+----------------------+----------------------+--------------------+ | |
| | | token (int64) | gen version (uint32) | topic length (uint8) | topic name (bytes) | | |
| | +---------------+----------------------+----------------------+--------------------+ | |
| | | compressed payload (zstd crc) (bytes)                                            | | |
| | +----------------------------------------------------------------------------------+ | |
| +--------------------------------------------------------------------------------------+ |
|                                                                                          |
| +--------------------------------------------------------------------------------------+ |
| | item                                                                                 | |
| +--------------------------------------------------------------------------------------+ |
|                                                                                          |
| .                                                                                        |
| .                                                                                        |
| .                                                                                        |
|                                                                                          |
+------------------------------------------------------------------------------------------+
```
